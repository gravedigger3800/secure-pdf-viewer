# SecureView PDF (MVP)

A Proof-of-Concept (POC) for a secure, ephemeral PDF sharing tool. This application allows users to generate time-limited, view-only links to PDF documents with built-in deterrents against downloading and printing.

## üöÄ Features

*   **Time-Limited Access**: Links expire automatically after 1 hour.
*   **Viewer Only**: No download button, print disabled (via CSS and JS), right-click disabled.
*   **Dynamic Watermarking**: Overlays timestamp and status on the document.
*   **Security Pause**: Blurs/Hides the document if the user switches tabs or takes focus away.
*   **Two Operation Modes**:
    1.  **Local Test (Browser Storage)**: Upload a file, saved to `localStorage`. Link works *only* on the host device.
    2.  **Public URL (Sharing)**: Wrap an external public URL (Dropbox/S3/Website). Link works for *anyone*.

## üìÇ Project Structure

*   `src/components/AdminPanel.tsx`: The dashboard to upload files or input URLs to generate links.
*   `src/components/PdfViewer.tsx`: The secure viewing interface with watermarks and protection logic.
*   `src/utils/security.ts`: Hooks for disabling context menus and handling page visibility (tab switching).
*   `src/services/storageService.ts`: Handles the mock "database" using browser LocalStorage.

## ‚ö†Ô∏è Important Notes for Deployment

*   **No Backend**: This is a static Single Page Application (SPA).
